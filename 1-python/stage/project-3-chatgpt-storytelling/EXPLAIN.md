# EXPLAIN: ChatGPT Storytelling (No Code)

## What This Project Is

You use ChatGPT in a chat window. You don't write any code. You type prompts and choices. ChatGPT writes the story and gives you options. You pick, and the story continues. It's like a choose-your-own-adventure book, but the AI writes it live.

## Key Idea

**Prompt engineering** = How you ask the AI determines what you get. A good prompt tells ChatGPT exactly what you want: "Give me 2-3 choices at the end." A vague prompt gets vague results.

## For Teaching

Students learn that AI is a tool for creativity. They see how the way you ask changes the output. No programming needed—anyone can do it.
