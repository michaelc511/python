# TEACH: Script for Teaching This Project

## Opening

"This project has zero code. You're going to chat with ChatGPT and build a story together. You make choices. ChatGPT writes what happens next."

## Demo

1. Open ChatGPT.
2. Paste the starter prompt from STORY_PROMPTS.md.
3. Read ChatGPT's scene aloud.
4. Pick a choice, type it in.
5. Repeat 2-3 times so they see the pattern.

## Key Lesson

"The magic is in the prompt. We told ChatGPT: give me choices, keep it short, start somewhere mysterious. It did. If we said 'write a story' with no instructions, we'd get something different. How you ask matters."

## Closing

"Use this with students who aren't coders yet. They'll see AI as a creative partner."
