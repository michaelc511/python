# CONCEPTS: What You Need to Know

## Interactive Story
A story where the reader (you) makes choices that change what happens next. Like a video game with text.

## Prompt
The text you type to tell the AI what you want. "Give me 2 choices" is a prompt.

## Prompt Engineering
The skill of writing prompts so the AI gives you exactly what you need. Clear instructions = better results.

## No Code
This project doesn't use Python or any programming. You use ChatGPT's website directly.
