# TEACH: Script for Teaching This Project

## Opening

"We're turning text into images. You describe it, DALL-E draws it. Gradio gives us a simple web form—no HTML to write."

## Demo

1. Run the app
2. Type: "A minimalist logo for a coffee shop"
3. Click Submit. Show the result.
4. Try a different prompt. "A robot watering plants in a garden."

## Key Idea

"Same pattern as the HR chatbot: we have a function that does the work. Gradio wraps it in a UI. Input → function → output. That's it."

## Closing

"Designers use this for mockups and inspiration. Marketing uses it for campaign visuals. The prompt is the skill—be specific."
