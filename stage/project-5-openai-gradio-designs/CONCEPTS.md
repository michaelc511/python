# CONCEPTS: What You Need to Know

## DALL-E
OpenAI's model that creates images from text. You send a description, it returns an image.

## API
Application Programming Interface. A way for your code to talk to OpenAI's servers. You send a request, they send back a response.

## Gradio Interface
gr.Interface(fn=your_function, inputs=[...], outputs=[...])—wraps your function in a web form. Inputs can be text, dropdown, slider. Outputs can be image, text, etc.
