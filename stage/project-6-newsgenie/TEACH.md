# TEACH: Script for Teaching This Project

## Opening

"NewsGenie does two things: gives you news and answers general questions. One app, two brains. We decide which to use by looking at what the user said."

## Key Idea: Query Routing

"We don't send everything to GPT. If they want news, we call the news API. If they want a general answer, we call GPT. Simple if/else based on keyword check."

## Demo

1. Ask: "What's the latest technology news?" → Show news output
2. Ask: "What is machine learning?" → Show GPT output
3. Ask: "Sports headlines" → Show sports news

## Closing

"This pattern—route to different services based on user intent—is common. Feedback bot vs. question bot. Support vs. sales. Same idea."
